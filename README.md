# 🚀 Initializer

> **System Security Initialization** - Initializes security baseline and system hardening on startup.

---

## 📋 Overview

Initializer sets up the security environment on system startup, applying baselines, checking configurations, and preparing the system for secure operation.

---

## 🎯 What It Does

- 🚀 **System Init** - Security initialization
- 📊 **Baseline Check** - Verify security baseline
- ⚙️ **Config Setup** - Initialize configurations
- 🛡️ **Hardening Apply** - Apply startup hardening

---

## 🚀 Usage

```powershell
# Run initialization
.\Initializer.ps1
```
